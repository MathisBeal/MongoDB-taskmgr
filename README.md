# Task Manager MongoDB
Mathis BEAL

## Lancer le projet
Installer les dépendances
```bash
pnpm install
```

Lancer le projet
```bash
pnpm start
```

### Installer pnpm si non installé
```
npm install -g pnpm
```

## Le projet

Le projet est accessible à l'addresse [http://localhost:3000](http://localhost:3000).
